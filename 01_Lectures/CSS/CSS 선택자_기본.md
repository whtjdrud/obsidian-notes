## \* 전체 선택자
---
```css
* {
	color: red;  
}
```
- 모든 요소를 선택한다.
## 태그 선택자(Type Selector)
---
```css
li {
	color: red;
}
```
- _**태그 이름**_ 이 ABC인 요소 선택

## 클래스 선택자(Class Selector)
---
```css
.orange {
	color: red;
}
```
- HTML **class 속성의 값**이 ABC인 요소 선택

## 아이디 선택자(ID Selector)
---
```css
#orange {
	color: red;
}
```
